# Reliance Digital Mobile Sales Analytics Dashboard

## 📌 Project Overview
This repository features an interactive, end-to-end **Power BI Dashboard** that analyzes multi-brand mobile sales performance across Reliance Digital retail footprints in India. Designed with comprehensive reporting frames and a dynamic UI/UX framework, this dashboard transforms raw retail data into highly actionable business intelligence to drive revenue, monitor market trends, and optimize inventory strategy.

---

## 🛠️ Tech Stack & Advanced Features
* **Power BI Desktop**: End-to-end dashboard design, data modeling, and data visualization.
* **DAX (Data Analysis Expressions)**: Formulated complex custom measures and calculated columns for advanced KPI tracking, sales performance, and dynamic filtering.
* **Data Transformation**: Cleaned, structured, and aggregated multi-company sales datasets to ensure absolute reporting accuracy.

---

## 📊 Key Metrics & Analytics Frames
* **Sales Performance**: Deep dive into revenue, unit sales, and growth trends across various mobile brands.
* **Geographical Insights**: Regional sales distribution across different store locations in India.
* **Brand & Model Analysis**: Comparative performance benchmarks identifying top-performing smartphone segments.
* **Interactive UI/UX**: Dynamic filtering frames by time period, brand, region, and price brackets for seamless stakeholder exploration.

---

## 🚀 How to Explore the Dashboard
1. Clone or download this repository.
2. Open the `PowerBI Mobile Sales Dashboard.pbix` file using **Power BI Desktop**.
3. Interact with the slicers and visual frames to uncover localized market insights.
